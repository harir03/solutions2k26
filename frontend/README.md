# IntelliCredit Dashboard

React/Next.js dashboard for the IntelliCredit AI-powered alternate credit scoring system.

## Getting Started

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Components

This dashboard uses components from the dashboard template:

- **StatCard** - Display key metrics
- **ProgressBarCard** - Show score distribution
- **DonutChartCard** - Visualize applicant breakdown
- **Header/Sidebar** - Navigation layout

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Recharts

## Deployment

Deploy on Vercel:

```bash
npm run build
vercel deploy
```
